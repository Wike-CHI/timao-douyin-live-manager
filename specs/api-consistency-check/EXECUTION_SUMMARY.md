## 🚨 高优先级问题（需立即修复）

### 1. PAY-001: Decimal vs number 类型不匹配
- **影响**: 金额精度丢失，可能导致财务错误
- **修复**: 前后端统一使用字符串传输金额
- **预计时间**: 4 小时

### 2. PAY-005: 创建支付请求参数不完整
- **影响**: 支付功能无法正常工作
- **修复**: 补全 amount、currency、coupon_code、return_url 等字段
- **预计时间**: 2 小时

### 3. AUTH-004: username 验证逻辑缺失
- **影响**: 可能提交无效用户名，被后端拒绝
- **修复**: 前端添加完整验证逻辑
- **预计时间**: 3 小时

### 4. MISC-004: API 路径重复
- **影响**: 维护复杂度高，代码冗余
- **修复**: 统一使用 /api/payment 或 /api/subscription
- **预计时间**: 6 小时

### 5. DY-003: 抖音状态响应缺少 fetcher_status
- **影响**: 无法获取抓取器详细状态
- **修复**: 前端类型定义添加 fetcher_status
- **预计时间**: 1 小时

### 6. AUDIO-003: 音频高级设置类型定义不完整
- **影响**: 类型不安全，容易出错
- **修复**: 定义完整的 LiveAudioAdvancedSettings 接口
- **预计时间**: 2 小时

### 7. PAY-004: 订阅响应日期时间类型批量问题
- **影响**: 文档不清晰，可能导致解析错误
- **修复**: 在 API 文档中明确说明日期时间格式
- **预计时间**: 1 小时

**总计修复时间**: **19 小时** (高优先级)
