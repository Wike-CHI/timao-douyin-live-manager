# 提猫直播助手移动端 - 用户体验分析

## 1. 用户体验分析

### 核心用户角色
- **主播**：需要实时查看弹幕、接收AI话术建议、查看直播数据分析
- **助手**：协助主播管理直播，监控数据，生成复盘报告

### 核心使用场景
1. **登录场景**：快速登录，支持记住密码
2. **直播监控场景**：实时查看弹幕和礼物，通过悬浮窗接收AI分析和话术建议
3. **数据复盘场景**：查看历史直播数据，手动生成完整复盘报告
4. **订阅管理场景**：查看套餐，支付升级

### 关键交互需求
- **实时性**：直播监控需要实时更新，WebSocket数据推送
- **便捷性**：悬浮窗快速显示AI建议，一键复制话术
- **可视化**：数据统计使用图表展示，直观易懂
- **易用性**：界面简洁，操作流程顺畅

```
提猫直播助手
├── 登录/注册
│   ├── 登录页面
│   └── 注册页面
```

### 核心界面定义

5. **设置页面**：个人设置和系统配置

## 3. UI设计规范

### 设计风格
- **设计语言**：遵循 iOS Human Interface Guidelines 和 Material Design
- **主色调**：#667eea（紫色品牌色）
- **辅助色**：#764ba2（深紫色）
- **背景**：浅色模式 #ffffff，深色模式 #121212

### 组件规范
- **卡片样式**：圆角 12px，阴影层级分明
- **按钮样式**：主要按钮使用品牌紫色，次要按钮使用灰色边框
- **悬浮窗**：半透明背景 rgba(102, 126, 234, 0.95)，圆角 12px
- **字体**：标题 20-24px，正文 14-16px，辅助文字 12px

## 4. 核心页面设计

### 页面1：登录页面
- 品牌Logo和应用名称
- 用户名/邮箱输入框
- 密码输入框（支持显示/隐藏）
- "记住我"复选框
- 登录按钮
- 注册链接

### 页面2：直播监控页面
- 顶部：直播状态指示器、观众数统计
- 中部：实时弹幕列表（滚动更新）
- 底部：控制按钮（开始/暂停/停止）
- 悬浮窗：AI分析、话术建议（可拖拽）

### 页面3：数据统计列表页面
- 顶部：日期筛选器
- 列表：直播记录卡片（时间、时长、数据概览、复盘状态）
- 刷新和加载更多

### 页面4：复盘详情页面
- 直播基本信息
- 数据统计图表
- AI分析内容
- 深度分析和建议
- 生成复盘按钮
