## 🚀 Cursor连接云服务器（推荐）

### 方式1: Remote-SSH扩展（最推荐）⭐

Cursor可以直接通过Remote-SSH扩展连接到云服务器！

#### 步骤1: 安装扩展

```
1. 打开Cursor
2. 按 Ctrl+Shift+X 打开扩展
3. 搜索 "Remote - SSH"
4. 点击安装
```

#### 步骤2: 配置SSH

创建或编辑 `C:\Users\你的用户名\.ssh\config`：

```
Host timao-server
    HostName 你的服务器IP
    User root
    Port 22
    IdentityFile C:\Users\你的用户名\.ssh\id_rsa
```

#### 步骤3: 连接服务器

```
1. 按 F1 打开命令面板
2. 输入 "Remote-SSH: Connect to Host"
3. 选择 "timao-server"
4. 等待连接完成
```

#### 步骤4: 在服务器上部署

连接成功后，在Cursor终端中执行：

```bash
# 克隆项目（首次）
cd /opt
git clone https://github.com/your-repo/timao-douyin-live-manager.git
cd timao-douyin-live-manager

# 配置环境
cd server
cp env.production.template .env
vim .env  # 填写数据库等配置

# 部署服务
cd ..
docker-compose -f docker-compose.full.yml up -d --build
```

**优势**：
- ✅ 直接在服务器上操作
- ✅ 无需传输文件
- ✅ 开发体验最好
- ✅ 适合持续开发和维护

详细说明：[deploy/WINDOWS_DEPLOY.md](deploy/WINDOWS_DEPLOY.md)

## 🎨 三种部署流程对比

```

Remote-SSH扩展连接

直接在服务器上操作
```

## ⚙️ 必需配置

### 服务器配置（上传方式需要）

文件：`deploy/upload_config.env`

```env
SERVER_HOST="123.45.67.89"       # 服务器IP
SERVER_USER="root"                # SSH用户名
SERVER_PATH="/opt/timao-douyin"   # 部署路径
```

### 应用配置

文件：服务器上的 `server/.env`

```env
# 后端端口
BACKEND_PORT=11111

# 数据库配置
MYSQL_HOST=数据库地址
MYSQL_PORT=3306
MYSQL_USER=timao
MYSQL_PASSWORD=你的密码
MYSQL_DATABASE=timao

# 安全配置
SECRET_KEY=自动生成或手动设置
CORS_ORIGINS=*

# 日志级别
LOG_LEVEL=INFO
```

## 🔑 SSH密钥配置（推荐）

### 生成密钥

```bash
# Windows PowerShell / Git Bash / Linux / Mac
ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
```

### 上传到服务器

```bash
# 方法1（Linux/Mac/WSL）
ssh-copy-id root@服务器IP

# 方法2（Windows/通用）
type $env:USERPROFILE\.ssh\id_rsa.pub | ssh root@服务器IP "cat >> ~/.ssh/authorized_keys"
```

### 测试连接

```bash
ssh root@服务器IP
```

免密登录成功即配置完成！

## 🚨 常见问题

### Q: Cursor能直接连接云服务器吗？

**A: 可以！** 使用Remote-SSH扩展即可。这是最推荐的方式。

### Q: Windows环境如何部署？

**A: 三种方式：**
1. **Remote-SSH**（推荐）- 直接连接服务器
2. **PowerShell脚本** - `.\deploy\deploy.ps1`
3. **Git Bash** - `./deploy/deploy_all.sh`

### Q: 不想安装WSL可以吗？

**A: 可以！** 使用Remote-SSH或Git Bash即可。

### Q: 如何查看部署状态？

```bash
# 方法1: 直接SSH
ssh root@服务器IP 'docker-compose -f /opt/timao-douyin/docker-compose.full.yml ps'

# 方法2: Remote-SSH连接后查看
docker-compose -f docker-compose.full.yml ps
```

### Q: 外部无法访问？

开放防火墙端口：

```bash
sudo ufw allow 80/tcp
sudo ufw allow 11111/tcp
```
