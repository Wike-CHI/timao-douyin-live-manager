## 📋 部署步骤

### 1. 部署后端到服务器

#### 方法1: 使用部署脚本（推荐）

```bash
# 给脚本执行权限
chmod +x deploy-server.sh

# 执行部署
./deploy-server.sh
```

#### 方法2: 手动部署

```bash
# 1. SSH连接到服务器
ssh root@129.211.218.135

# 2. 创建项目目录
mkdir -p /opt/timao-douyin-live-manager
cd /opt/timao-douyin-live-manager

# 3. 克隆或上传代码
git clone <your-repo-url> .

# 4. 安装依赖
cd server
pip install -r requirements.txt

# 5. 配置环境变量
export BACKEND_PORT=10050
export DEBUG=false

# 6. 启动服务
python -m uvicorn app.main:app --host 0.0.0.0 --port 10050
```

#### 使用systemd管理服务

```bash
# 启动服务
sudo systemctl start timao-backend

# 开机自启
sudo systemctl enable timao-backend

# 查看状态
sudo systemctl status timao-backend

# 查看日志
sudo journalctl -u timao-backend -f
```

## 🐛 常见问题

### 问题1: 后端无法访问

**检查项**:
1. 防火墙是否开放端口: `sudo ufw allow 10050/tcp`
2. 服务是否运行: `sudo systemctl status timao-backend`
3. 服务监听地址: 应该是 `0.0.0.0`，不是 `127.0.0.1`

### 问题2: Electron应用连接失败

**检查项**:
1. 后端服务是否可访问: `curl http://129.211.218.135:10050/health`
2. CORS配置是否正确（后端需要允许Electron应用访问）
3. 检查 `apiConfig.ts` 中的默认地址是否正确

### 问题3: CORS错误

**解决**: 确保后端配置了CORS，允许所有来源:

```python
# server/app/main.py
from fastapi.middleware.cors import CORSMiddleware

app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],  # 生产环境建议限制域名
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)
```
