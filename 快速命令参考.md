## 🔧 手动服务管理

### 停止服务
```bash
pkill -f "python.*server/main.py"
pkill -f "python.*server/streamcap_service.py"
pkill -f "python.*server/douyin_service.py"
```

### 启动服务（后台）
```bash
cd /www/wwwroot/wwwroot/timao-douyin-live-manager
source venv/bin/activate

nohup python -m uvicorn server.main:app --host 0.0.0.0 --port 8000 > logs/main.log 2>&1 &
nohup python -m uvicorn server.streamcap_service:app --host 0.0.0.0 --port 8001 > logs/streamcap.log 2>&1 &
nohup python -m uvicorn server.douyin_service:app --host 0.0.0.0 --port 8002 > logs/douyin.log 2>&1 &
```

### 启动服务（前台调试）
```bash
cd /www/wwwroot/wwwroot/timao-douyin-live-manager
source venv/bin/activate

# 在不同终端分别运行
python -m uvicorn server.main:app --host 0.0.0.0 --port 8000
python -m uvicorn server.streamcap_service:app --host 0.0.0.0 --port 8001
python -m uvicorn server.douyin_service:app --host 0.0.0.0 --port 8002
```

## 🔄 Nginx管理

```bash
# 测试配置
nginx -t

# 重启Nginx
systemctl restart nginx

# 查看Nginx状态
systemctl status nginx

# 重新加载配置（不中断服务）
nginx -s reload
```

## 📦 使用Supervisor（推荐）

### 首次安装
```bash
chmod +x setup-supervisor.sh
./setup-supervisor.sh
```

### 日常使用
```bash
# 查看所有服务状态
supervisorctl status

# 启动所有服务
supervisorctl start timao-services:*

# 停止所有服务
supervisorctl stop timao-services:*

# 重启所有服务
supervisorctl restart timao-services:*

# 重启单个服务
supervisorctl restart timao-main
supervisorctl restart timao-streamcap
supervisorctl restart timao-douyin

# 查看实时日志
supervisorctl tail -f timao-main
```

## 🛠️ 常见问题

### 端口被占用
```bash
# 查看占用端口的进程
netstat -tulnp | grep :8181   # 后端端口
netstat -tulnp | grep :10050  # 前端端口
lsof -i :8181
lsof -i :10050

# 杀掉进程
kill -9 <PID>
```
