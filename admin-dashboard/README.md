# 管理后台系统

基于 react-admin 构建的提猫直播助手管理后台。

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

访问：http://localhost:3000

### 构建

```bash
npm run build
```

## 环境变量

创建 `.env` 文件（可选）：

```env
VITE_FASTAPI_URL=http://127.0.0.1:9019
```
