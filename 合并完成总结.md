## 📝 未来合并指南

### 下次合并其他分支时

1. **自动保护的文件**（无需担心）
   - `.env` 文件会自动保留本地版本
   - 配置文件会自动保留本地版本
   - 运行脚本会自动保留本地版本

2. **需要手动处理的文件**
   - 代码文件（如 `electron/main.js`）
   - 前端组件
   - 后端服务

3. **合并步骤**
   ```bash
   # 1. 切换到目标分支
   git checkout local-test
   
   # 2. 合并源分支
   git merge <源分支名>
   
   # 3. 如果有冲突
   # - 配置文件会自动保留本地版本 ✅
   # - 代码文件需要手动解决
   
   # 4. 解决冲突后
   git add <冲突文件>
   git commit -m "merge: 合并分支"
   ```
