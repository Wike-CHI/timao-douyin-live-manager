## 🔧 故障排查

### 问题1：环境变量未加载

**错误**: `XUNFEI_API_KEY 未设置`

**解决**:
1. 确认 `server/.env` 文件存在
2. 确认文件内容格式正确
3. 确认文件路径正确

### 问题2：API Key格式错误

**错误**: `XUNFEI_API_KEY 格式错误`

**解决**:
1. 确认格式为 `APPID:APISecret`（用冒号分隔）
2. 确认没有多余空格
3. 确认APPID和APISecret都填写了

### 问题3：网络连接失败

**错误**: `Connection timeout` 或 `Connection refused`

**解决**:
1. 检查网络连接
2. 确认可以访问 `https://spark-api-open.xf-yun.com`
3. 检查防火墙设置

### 问题4：认证失败

**错误**: `401 Unauthorized` 或 `Invalid API Key`

**解决**:
1. 确认APPID和APISecret正确
2. 确认API Key格式正确（APPID:APISecret）
3. 确认账户有权限使用星火大模型服务

### 问题5：模型不存在

**错误**: `Model 'xxx' not found`

**解决**:
1. 确认模型名称正确：`lite`、`generalv3`、`generalv3.5`、`4.0Ultra`
2. 确认账户有权限使用该模型
3. 使用免费的 `lite` 模型测试
