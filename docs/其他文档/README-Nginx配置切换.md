3. 确认无误后，重载 Nginx
4. 访问公网IP验证效果
5. 如果有问题，立即恢复备份

### 回滚方案

如需恢复前端显示，修改 `location /` 块：

```nginx
location / {
    root /www/wwwroot/timao-app;
    index index.html;
    try_files $uri $uri/ /index.html;
}
```

## 🤝 技术支持

### 遇到问题？

1. **查看故障排查文档**: `docs/Nginx配置故障排查.md`
2. **检查后端服务**: `bash scripts/部署与运维/check-backend-status.sh`

4. **验证配置语法**: `nginx -t`

### 常见问题

| 问题 | 解决方案 |
|------|----------|
| 502 Bad Gateway | 检查后端服务是否运行 |
| 仍显示旧页面 | 清除浏览器缓存（Ctrl+Shift+R） |
| 配置保存失败 | 检查配置语法是否正确 |
| API无法访问 | 验证代理配置和端口 |
