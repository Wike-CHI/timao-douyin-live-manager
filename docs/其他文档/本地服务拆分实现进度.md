## 🔧 使用方式

### 开发环境
```bash
# 云端服务
cd server/cloud
python main.py  # 监听 15000

# 本地服务（在 Electron 中自动启动）
cd server/local
python main.py  # 监听 16000（仅本地）

# Electron
npm run dev
```

### 生产环境
- 云端：PM2 部署 `server/cloud/main.py`
- 本地：打包进 Electron 安装包,首次运行自动下载模型
