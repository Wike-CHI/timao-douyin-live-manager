## 📊 模式对比表

| 对比项 | 前端显示模式 | 后端显示模式（新） |
|--------|-------------|------------------|
| **根路径 `/`** | 静态文件 (timao-app) | 代理到后端 (11111) |
| **页面内容** | React 构建的前端应用 | 后端服务状态页 |
| **背景样式** | 前端自定义样式 | 紫色渐变背景 + 猫咪图标 |
| **API访问** | 需单独配置 `/api/` | 自动包含所有路由 |
| **静态资源** | 前端 assets 目录 | 后端 /static 目录 |
| **WebSocket** | 需单独配置 | 自动支持 |
| **文档访问** | 可能需配置 | `/docs` 直接可用 |
| **适用场景** | 完整 Web 应用 | API 服务 + 状态展示 |

## 🔧 配置代码对比

### 前端显示模式配置

```nginx
server {
    listen 80;
    server_name 129.211.218.135;
    
    # 根路径 - 静态文件
    location / {
        root /www/wwwroot/timao-app;
        index index.html;
        try_files $uri $uri/ /index.html;
    }
    
    # API 需单独配置
    location /api/ {
        proxy_pass http://127.0.0.1:11111/api/;
        # ... 代理配置
    }
    
    # 文档需单独配置
    location /docs {
        proxy_pass http://127.0.0.1:11111/docs;
        # ... 代理配置
    }
}
```

### 后端显示模式配置（新）

```nginx
server {
    listen 80;
    server_name 129.211.218.135;
    
    # 根路径 - 直接代理到后端
    location / {
        proxy_pass http://127.0.0.1:11111;
        proxy_http_version 1.1;
        
        # WebSocket 支持
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        
        # 请求头
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        
        # 禁用缓存
        proxy_buffering off;
        proxy_cache off;
    }
    
    # API、文档、静态资源都自动包含
    # 无需额外配置
}
```

    ↓

    ↓
返回 timao-app/index.html

    ↓

    ↓

## ⚡ 性能对比

### 前端显示模式

- ✅ 首屏加载快（静态文件）
- ✅ 前端交互丰富
- ❌ 需要打包构建
- ❌ 配置相对复杂

### 后端显示模式（新）

- ✅ 配置简单

- ✅ 维护方便
- ✅ 适合 API 服务

## 🎯 使用场景建议

### 选择前端显示模式

适用于：
- 需要完整的 Web 应用界面
- 有丰富的前端交互需求
- 用户需要在浏览器中操作各种功能
- 前端和后端完全分离

### 选择后端显示模式（新）

适用于：
- 主要提供 API 服务
- 仅需展示服务状态和文档
- 简化部署和维护
- 快速上线，无需前端构建

## 🔄 如何切换

### 从前端模式切换到后端模式

1. 备份当前配置
2. 使用 `nginx-config-backend-only.conf`
3. 重载 Nginx
4. 访问验证

### 从后端模式切换回前端模式

1. 恢复 `location /` 配置：
   ```nginx
   location / {
       root /www/wwwroot/timao-app;
       index index.html;
       try_files $uri $uri/ /index.html;
   }
   ```
2. 添加 `/api/` 代理配置
3. 重载 Nginx
