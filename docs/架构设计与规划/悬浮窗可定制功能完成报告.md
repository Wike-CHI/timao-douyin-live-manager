## 📋 新增功能

### 功能 #1: 钉子按钮切换置顶 ✅

**需求**:
- 用户提出："不强制显示在应用顶部，用户可以在窗口顶部有个钉子按钮调节"

**实现**:

#### 后端 (Electron主进程)

**文件**: `electron/floatingWindow.js`
```javascript
// 创建窗口时根据配置设置置顶状态
floatingWindow = new BrowserWindow({
  alwaysOnTop: config.alwaysOnTop,  // 🆕 根据配置决定
  // ...
});

// 🆕 切换置顶状态
function toggleAlwaysOnTop() {
  const currentState = floatingWindow.isAlwaysOnTop();
  const newState = !currentState;
  
  floatingWindow.setAlwaysOnTop(newState);
  saveFloatingConfig({ alwaysOnTop: newState, ... });
  
  return newState;
}

// 🆕 获取置顶状态
function getAlwaysOnTopState() {
  return floatingWindow.isAlwaysOnTop();
}
```

**改进**:
- ✅ 初始置顶状态可配置（默认为 `true`）
- ✅ 用户可随时切换置顶状态
- ✅ 置顶状态持久化到配置文件
- ✅ 下次启动自动恢复上次的置顶状态

---

#### IPC通信

**文件**: `electron/main.js`
```javascript
// 切换置顶状态
ipcMain.handle('toggle-floating-always-on-top', async () => {
  const newState = toggleAlwaysOnTop();
  return { success: true, alwaysOnTop: newState };
});

// 获取置顶状态
ipcMain.handle('get-floating-always-on-top', async () => {
  return getAlwaysOnTopState();
});
```

**文件**: `electron/preload.js`
```javascript
// 暴露给渲染进程的API
toggleFloatingAlwaysOnTop: () => ipcRenderer.invoke('toggle-floating-always-on-top'),
getFloatingAlwaysOnTop: () => ipcRenderer.invoke('get-floating-always-on-top'),
```

---

#### 前端 (React UI)

**文件**: `electron/renderer/src/pages/FloatingWindowPage.tsx`
```typescript
// 状态管理
const [alwaysOnTop, setAlwaysOnTop] = useState<boolean>(true);

// 初始化时获取状态
useEffect(() => {
  window.electronAPI.getFloatingAlwaysOnTop().then(state => {
    setAlwaysOnTop(state);
  });
}, []);

// 切换处理
const handleToggleAlwaysOnTop = async () => {
  const result = await window.electronAPI.toggleFloatingAlwaysOnTop();
  if (result.success) {
    setAlwaysOnTop(result.alwaysOnTop);
  }
};

// 🆕 钉子按钮 UI
<button
  onClick={handleToggleAlwaysOnTop}
  title={alwaysOnTop ? '取消置顶' : '窗口置顶'}
  style={{
    background: alwaysOnTop ? 'rgba(168, 85, 247, 0.2)' : 'transparent',
    color: alwaysOnTop ? '#a855f7' : '#94a3b8',
  }}
>
  📌
</button>
```

**UI特点**:
- ✅ 钉子图标 (📌)
- ✅ 置顶时：紫色高亮背景
- ✅ 未置顶时：灰色透明背景
- ✅ 鼠标悬停：高亮效果
- ✅ Tooltip提示："取消置顶" / "窗口置顶"

### 功能 #2: 可调整窗口大小 ✅

**需求**:
- 用户提出："用户不只能调节位置，也能调整悬浮窗大小以方便查看信息"

**实现**:

#### Electron窗口配置

**文件**: `electron/floatingWindow.js`
```javascript
floatingWindow = new BrowserWindow({
  width: config.width,      // 🆕 从配置加载宽度
  height: config.height,    // 🆕 从配置加载高度
  minWidth: 280,            // 🆕 最小宽度
  minHeight: 240,           // 🆕 最小高度
  maxWidth: 600,            // 🆕 最大宽度
  maxHeight: 800,           // 🆕 最大高度
  resizable: true,          // 🆕 可调整大小
  // ...
});

// 🆕 监听窗口大小变化
floatingWindow.on('resize', () => {
  saveFloatingConfig({
    width: bounds.width,
    height: bounds.height,
    // ...
  });
});
```

**尺寸限制**:
| 属性 | 值 | 说明 |
|------|-----|------|
| **默认宽度** | 320px | 适合大部分信息显示 |
| **默认高度** | 280px | 足够显示主要内容 |
| **最小宽度** | 280px | 防止太小看不清 |
| **最小高度** | 240px | 保证基本可用性 |
| **最大宽度** | 600px | 防止太大遮挡主播视野 |
| **最大高度** | 800px | 适合竖屏显示器 |

---

#### 原生窗口拖拽边缘

**特性**:
- ✅ 四个角落可调整大小（对角拖拽）
- ✅ 四条边可调整宽度或高度
- ✅ 拖拽边缘厚度：自适应（Electron原生）
- ✅ 拖拽时窗口内容实时响应
- ✅ 松开鼠标后自动保存新尺寸

**用户体验**:
- 主播可根据需要放大查看AI建议
- 可缩小节省屏幕空间
- 尺寸记忆，下次启动保持

### 功能 #3: 配置持久化 ✅

**配置文件**: `config/floating-position.json`

**配置结构**:
```json
{
  "x": 1580,              // 窗口X坐标
  "y": 820,               // 窗口Y坐标
  "width": 400,           // 🆕 窗口宽度
  "height": 500,          // 🆕 窗口高度
  "alwaysOnTop": false    // 🆕 置顶状态
}
```

**持久化时机**:
| 事件 | 保存内容 | 防抖时间 |
|------|----------|----------|
| 窗口移动 | x, y, width, height, alwaysOnTop | 300ms |
| 窗口调整大小 | x, y, width, height, alwaysOnTop | 300ms |
| 切换置顶 | 立即保存所有配置 | 无 |
| 边缘吸附 | x, y, width, height, alwaysOnTop | 300ms |

**配置加载**:
- ✅ 启动时自动加载上次的配置
- ✅ 验证配置的有效性（屏幕内、尺寸合理）
- ✅ 无效配置时使用默认值
- ✅ 兼容旧版本配置（只有x,y）
