## 🔧 技术实现

### 1. 服务器端：只保存音频流

**文件位置**：`server/app/services/live_audio_stream_service.py`

**核心逻辑**：ffmpeg多输出

```python
# ffmpeg同时输出到两个目标：
# 1. pipe:1 → 实时转写（讯飞ASR）
# 2. live_audio.wav → 保存到文件（复盘用）

ffmpeg -i 直播流 \
  -map 0:a -ac 1 -ar 16000 -f s16le pipe:1 \  # 实时转写
  -map 0:a -ac 1 -ar 16000 -f wav live_audio.wav  # 保存文件
```

**保存路径**：
```
records/sessions/{session_id}/live_audio.wav
```

**音频格式**：
- 格式：WAV
- 采样率：16kHz
- 声道：单声道（Mono）
- 比特率：16bit PCM

**文件大小**（估算）：
```
1小时直播 ≈ 60 MB
2小时直播 ≈ 120 MB
```

#### 2. 下载音频文件

```typescript
async function downloadAudio(audioPath: string) {
  const serverUrl = `http://129.211.218.135/records/...`;
  const localPath = path.join(app.getPath('documents'), 'TalkingCat', 'audio', 'live_audio.wav');
  
  // 使用axios下载
  const response = await axios.get(serverUrl, { responseType: 'stream' });
  const writer = fs.createWriteStream(localPath);
  response.data.pipe(writer);
  
  return new Promise((resolve, reject) => {
    writer.on('finish', resolve);
    writer.on('error', reject);
  });
}
```

#### 3. Nginx静态文件服务（需要配置）

#### 4. 服务器自动清理（定时任务）

```python
# server/tasks/cleanup_audio_files.py
async def cleanup_old_audio_files():
    """清理3天前的临时音频文件"""
    import time
    import shutil
    
    records_dir = Path("records/sessions")
    cutoff_time = time.time() - (3 * 24 * 3600)  # 3天
    
    for session_dir in records_dir.glob("*"):
        audio_file = session_dir / "live_audio.wav"
        if audio_file.exists():
            if audio_file.stat().st_mtime < cutoff_time:
                audio_file.unlink()
                logger.info(f"清理旧音频: {audio_file}")
```

   ```typescript
   const { spawn } = require('child_process');
   
   function startLocalRecording(streamUrl: string, savePath: string) {
     const ffmpegPath = path.join(process.resourcesPath, 'ffmpeg.exe');
     const process = spawn(ffmpegPath, [
       '-i', streamUrl,
       '-vn',           // 不要视频
       '-ac', '1',      // 单声道
       '-ar', '16000',  // 16kHz
       '-f', 'wav',     // WAV格式
       savePath
     ]);
     
     return process;
   }
   ```
