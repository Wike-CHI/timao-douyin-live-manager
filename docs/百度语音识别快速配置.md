根据[百度智能云语音识别文档](https://cloud.baidu.com/doc/SPEECH/s/Tl9mh38eu)，**实时语音识别-websocket API** 是最适合直播场景的接口：

### 📊 百度语音识别接口对比

| 接口类型 | 适用场景 | 是否适合直播 | 说明 |
|---------|---------|-------------|------|
| 短语音识别 | 60秒内短音频 | ❌ | 适合语音搜索、语音指令 |
| **实时语音识别-websocket** | **实时转写、语音助手** | ✅ **推荐** | **支持长时间流式识别** |
| 音频文件转写 | 离线批量处理 | ❌ | 适合会议录音、质检 |

### 🎯 实时语音识别的优势

- ✅ **WebSocket长连接** - 持续的音频流传输
- ✅ **流式识别** - 边说边转，实时返回结果
- ✅ **低延迟** - 适合直播场景
- ✅ **无时长限制** - 不受60秒限制
- ✅ **支持VAD** - 自动断句

## 📝 快速配置步骤（5分钟）

### 1️⃣ 开通百度语音识别服务

访问：https://console.bce.baidu.com/ai/#/ai/speech/overview/index

1. 登录百度智能云账号（没有则注册）
2. 点击"立即使用"或"创建应用"
3. 选择"语音识别"服务

### 2️⃣ 创建应用获取密钥

访问：https://console.bce.baidu.com/ai/#/ai/speech/app/list

1. 点击"创建应用"
2. 填写应用信息：
   - **应用名称**：timao-live-asr（自定义）
   - **应用描述**：提猫直播语音识别（可选）
   - **接口选择**：勾选"语音识别"
3. 创建成功后，查看应用详情：
   - **AppID**：应用ID
   - **API Key**：API密钥
   - **Secret Key**：密钥

⚠️ **重要**：妥善保存这三个密钥！

### 3️⃣ 修改配置文件

编辑 `/www/wwwroot/wwwroot/timao-douyin-live-manager/server/.env`：

```bash
# 🆓 百度ASR配置（推荐）
BAIDU_ASR_APP_ID=你的AppID
BAIDU_ASR_API_KEY=你的APIKey
BAIDU_ASR_SECRET_KEY=你的SecretKey
USE_BAIDU_ASR=1

# 禁用其他ASR
USE_ALIYUN_ASR=0
USE_IFLYTEK_ASR=0
```

### 4️⃣ 重启服务

```bash
cd /www/wwwroot/wwwroot/timao-douyin-live-manager
pkill -f "uvicorn.*11111"
nohup .venv/bin/python -m uvicorn server.app.main:app \
  --host 0.0.0.0 --port 11111 --workers 1 > nohup.out 2>&1 &
```

### 5️⃣ 验证配置

查看日志：
```bash
tail -f nohup.out | grep -i "百度"
```

应该看到：

## 🔧 技术细节

### 使用的接口

**实时语音识别-websocket API**
- 接口地址：`wss://vop.baidu.com/realtime_asr`
- 鉴权方式：access_token
- 支持格式：pcm/wav/opus/amr
- 采样率：8000/16000
- 识别模型：1537(普通话) / 1737(英语) / 1637(粤语)

### 配置参数

```python
# 音频格式
format: pcm          # 音频格式
rate: 16000          # 采样率（16kHz）
channel: 1           # 单声道

# 识别参数
dev_pid: 1537        # 普通话模型
vad: dnn             # VAD模式
max_punc_delay: 500  # 标点延迟(ms)
```

## 🐛 常见问题

### Q1: 提示"Invalid token"
**A**: 
1. 检查API Key和Secret Key是否正确
2. 检查应用是否已开通语音识别服务
3. 稍等5分钟，配置可能需要生效时间

### Q2: 连接失败或超时
**A**:
1. 检查网络连接：`ping vop.baidu.com`
2. 检查防火墙是否拦截WebSocket连接
3. 确认服务器时间是否正确

### Q3: 识别结果不准确
**A**:
1. 确认音频采样率为16kHz
2. 检查音频格式是否为PCM
3. 尝试调整VAD参数
4. 检查是否有背景噪音

### Q4: 如何查看用量？
**A**: 
访问百度智能云控制台 → 语音识别 → 用量统计
https://console.bce.baidu.com/ai/#/ai/speech/overview/index

## ⚙️ 高级配置

### 1. 切换识别模型

编辑 `server/modules/ast/baidu_asr_adapter.py`：

```python
@dataclass
class BaiduASRConfig:
    # 识别模型
    dev_pid: int = 1537  # 修改这里
    # 1537 = 普通话(纯中文识别)
    # 1737 = 英语
    # 1637 = 粤语
```

### 2. 调整VAD参数

```python
@dataclass
class BaiduASRConfig:
    vad: str = "dnn"  # VAD模式
    # dnn = 深度学习VAD（推荐）
    # touch = 触发式VAD
```

### 3. 标点符号延迟

```python
@dataclass
class BaiduASRConfig:
    max_punc_delay: int = 500  # 标点延迟(ms)
    # 值越小，标点出现越快，但可能不准确
    # 值越大，标点更准确，但延迟更高
```

## 🔄 多服务商配置方案

**推荐配置**：同时配置多个服务商，按需切换

```bash
# .env 配置示例
# 优先级：百度 > 阿里云 > 腾讯云

# 百度（主力）
USE_BAIDU_ASR=1
BAIDU_ASR_APP_ID=xxx
BAIDU_ASR_API_KEY=xxx
BAIDU_ASR_SECRET_KEY=xxx

# 阿里云（备用1）
USE_ALIYUN_ASR=0
ALIYUN_ASR_APP_KEY=xxx
ALIYUN_ACCESS_KEY_ID=xxx
ALIYUN_ACCESS_KEY_SECRET=xxx

# 腾讯云（备用2）
USE_TENCENT_ASR=0
TENCENT_SECRET_ID=xxx
TENCENT_SECRET_KEY=xxx
TENCENT_APP_ID=xxx
```

当百度额度用完后，手动修改：
```bash
USE_BAIDU_ASR=0
USE_ALIYUN_ASR=1
```
