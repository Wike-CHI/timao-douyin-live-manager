## ⚠️ 端口冲突问题

### 问题1：前端端口不一致
- **Vite 配置**: 10089
- **Electron 期望**: 10050  
- **实际运行**: 10051 (因为 10050 被后端占用)

### 问题2：端口被占用
```bash
# 10050 被后端占用
$ lsof -i :10050
python  405733  root  15u  IPv4  TCP *:10050 (LISTEN)

# 前端被迫使用 10051
$ lsof -i :10051
node    405922  root  26u  IPv4  TCP *:10051 (LISTEN)
```

## ✅ 建议的端口配置

### 方案1：分离端口（推荐）
```bash
# 后端
BACKEND_PORT=10050

# 前端开发服务器
FRONTEND_PORT=10089  # 修改 Electron 期望端口
```

### 方案2：统一端口
```bash
# 后端
BACKEND_PORT=11111

# 前端开发服务器
FRONTEND_PORT=10050
```
