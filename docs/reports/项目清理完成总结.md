## 清理脚本

已创建自动清理脚本: `scripts/辅助工具/clean-project.sh`

### 功能

1. ✅ 清理Python缓存
2. ✅ 清理日志文件（>10M或7天前）
3. ✅ 清理临时文件
4. ✅ 清理系统文件
5. ✅ 清理空目录
6. ✅ 清理npm缓存
7. ✅ 清理测试覆盖率文件
8. ⚙️ 可选：清理IDE缓存
9. ⚙️ 可选：清理Docker缓存
10. 🔍 检测大文件（>100M）

### 使用方法

```bash

chmod +x scripts/辅助工具/clean-project.sh

# 运行清理
./scripts/辅助工具/clean-project.sh
```

### 安全特性

- ✅ **保留虚拟环境** - `.venv/` 目录
- ✅ **保留依赖包** - `node_modules/`
- ✅ **保留版本控制** - `.git/`
- ✅ **保留模型文件** - AI模型不会被误删
- ✅ **可选清理** - IDE和Docker缓存需手动确认
- ✅ **大文件提醒** - 只提示，不自动删除

## Docker部署注意事项

### .dockerignore 配置

清理后更新了 `.dockerignore`，确保：

```dockerignore
# 排除但保留必要文件
__pycache__/
*.pyc
*.log

# 保留虚拟环境（Docker复用）
# .venv/  # 注释掉，保留

# 保留模型缓存
# server/models/.cache/  # 保留
```

### Docker镜像大小

预计Docker镜像大小：
- 基础镜像 (python:3.11-slim): ~500MB
- 虚拟环境 (.venv): ~6.8GB
- 项目代码: ~941MB
- **总计**: ~8.2GB

### 优化建议

如需进一步减小Docker镜像：

1. **多阶段构建** - 只复制必要的运行时文件
2. **精简依赖** - 移除开发依赖
3. **压缩镜像** - 使用 `docker-slim`
