# 云端本地混合部署架构设计

## 📋 架构概述

### 背景
- **问题**：服务器内存不足，无法支撑所有服务
- **解决方案**：将服务拆分为云端服务和本地服务
  - **云端**：用户系统、支付系统（轻量级，低内存占用）
  - **本地**：弹幕拉取、语音转写、AI处理等（重度服务，打包到客户端）

### 架构优势
✅ **降低服务器成本**：云端只运行轻量服务，内存需求 < 1GB  
✅ **提升用户体验**：本地服务响应更快，无网络延迟  
✅ **数据隐私保护**：敏感的直播数据不上传云端  
✅ **灵活扩展**：本地服务可使用用户的硬件资源

## 🔄 服务交互流程

### 1. 应用启动流程

```
用户启动应用
    ↓
Electron 主进程启动
    ↓
启动本地 FastAPI 服务 (16000端口)
    ├── 加载 SenseVoice 模型
    ├── 初始化 VAD 检测
    └── 启动 WebSocket 服务
    ↓
渲染进程加载
    ↓
检查登录状态
    ├── 未登录 → 调用云端登录 API (15000)
    └── 已登录 → 验证 Token (15000)
    ↓
进入主界面
    ├── 本地服务：弹幕拉取、语音转写
    └── 云端服务：用户信息、订阅状态
```

### 2. 用户登录流程

```
用户输入账号密码
    ↓
Electron → 云端 API (15000)
POST /api/auth/login
    ↓
云端验证账号密码
    ↓
返回 Token + 用户信息
    ↓
Electron 存储 Token
    ├── localStorage (渲染进程)
    └── electron-store (主进程)
    ↓
后续请求携带 Token
```

### 3. 语音转写流程

```
用户点击"开始转写"
    ↓
Electron → 本地 API (16000)
POST /api/transcribe/start
    ↓
本地服务加载模型
    ├── SenseVoice (已预加载)
    └── VAD 检测器
    ↓
接收音频流 (WebSocket)
    ↓
实时转写 → 返回文本
    ↓
Electron 显示转写结果
```

### 4. 订阅验证流程

```
用户使用高级功能
    ↓
Electron → 云端 API (15000)
GET /api/user/subscription
    ↓
云端验证订阅状态
    ├── 已订阅 → 返回权限
    └── 未订阅 → 返回受限
    ↓
Electron 根据权限控制功能
```

### Electron 集成本地服务

**main.js 启动本地服务**:
```javascript
// electron/main/index.js
const { spawn } = require('child_process');
const path = require('path');

let localAPIProcess = null;
const LOCAL_API_PORT = 16000;

// 启动本地 FastAPI 服务
async function startLocalAPI() {
  const pythonPath = process.env.PYTHON_PATH || 'python';
  const apiScript = path.join(__dirname, '../../server/local/main.py');
  
  localAPIProcess = spawn(pythonPath, [
    '-m', 'uvicorn',
    'server.local.main:app',
    '--host', '127.0.0.1',
    '--port', LOCAL_API_PORT.toString(),
    '--workers', '1'
  ]);

  localAPIProcess.stdout.on('data', (data) => {
    console.log(`[Local API] ${data}`);
  });

  localAPIProcess.stderr.on('data', (data) => {
    console.error(`[Local API Error] ${data}`);
  });

  // 等待服务启动
  await waitForService(`http://127.0.0.1:${LOCAL_API_PORT}/health`);
  console.log('✅ 本地 API 服务启动成功');
}

// 等待服务就绪
function waitForService(url, maxRetries = 30) {
  return new Promise((resolve, reject) => {
    let retries = 0;
    const interval = setInterval(async () => {
      try {
        const response = await fetch(url);
        if (response.ok) {
          clearInterval(interval);
          resolve();
        }
      } catch (error) {
        retries++;
        if (retries >= maxRetries) {
          clearInterval(interval);
          reject(new Error('本地服务启动超时'));
        }
      }
    }, 1000);
  });
}

// 应用启动
app.whenReady().then(async () => {
  // 1. 启动本地服务
  await startLocalAPI();
  
  // 2. 创建窗口
  createWindow();
});

// 应用退出时关闭本地服务
app.on('quit', () => {
  if (localAPIProcess) {
    localAPIProcess.kill();
  }
});
```

### 前端 API 调用

**API 配置**:
```typescript
// electron/renderer/src/config/api.ts
export const API_CONFIG = {
  // 云端服务（用户、支付）
  CLOUD_API: import.meta.env.VITE_CLOUD_API || 'https://your-server.com:15000',
  
  // 本地服务（直播、转写）
  LOCAL_API: 'http://127.0.0.1:16000',
};

// API 客户端
export class APIClient {
  // 云端请求
  static async cloudRequest(endpoint: string, options = {}) {
    const token = localStorage.getItem('token');
    return fetch(`${API_CONFIG.CLOUD_API}${endpoint}`, {
      ...options,
      headers: {
        'Authorization': `Bearer ${token}`,
        'Content-Type': 'application/json',
        ...options.headers,
      },
    });
  }

  // 本地请求
  static async localRequest(endpoint: string, options = {}) {
    return fetch(`${API_CONFIG.LOCAL_API}${endpoint}`, {
      ...options,
      headers: {
        'Content-Type': 'application/json',
        ...options.headers,
      },
    });
  }
}
```

**使用示例**:
```typescript
// 登录（云端）
async function login(username: string, password: string) {
  const response = await APIClient.cloudRequest('/api/auth/login', {
    method: 'POST',
    body: JSON.stringify({ username, password }),
  });
  const data = await response.json();
  localStorage.setItem('token', data.token);
  return data;
}

// 开始转写（本地）
async function startTranscribe() {
  const response = await APIClient.localRequest('/api/transcribe/start', {
    method: 'POST',
  });
  return response.json();
}
```

## 🔐 安全考虑

### 1. Token 管理
- **云端 Token**：用于云端 API 鉴权，存储在 localStorage
- **刷新机制**：Token 过期自动刷新
- **安全存储**：敏感信息使用 electron-store 加密存储

### 2. 本地服务安全
- **端口绑定**：只绑定 127.0.0.1，不允许外部访问
- **进程隔离**：本地服务只响应本地请求
- **数据加密**：敏感配置加密存储

### 3. 通信安全
- **HTTPS**：云端 API 使用 HTTPS
- **CORS**：严格限制跨域访问
- **请求签名**：关键请求添加签名验证

## 📝 注意事项

1. **模型文件大小**：SenseVoice 模型约 2.3GB，打包后安装包会很大
   - 解决方案：首次启动时下载模型
   
2. **Python 运行时**：需要打包 Python 环境
   - 解决方案：使用 pyinstaller 或打包精简版 Python

3. **跨平台兼容**：Windows/Mac/Linux 路径不同
   - 解决方案：使用 path.join 处理路径

4. **服务启动时间**：模型加载需要时间
   - 解决方案：显示加载进度，优化用户体验

5. **版本更新**：本地服务更新需要重新打包
   - 解决方案：实现热更新机制
