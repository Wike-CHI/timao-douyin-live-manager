# 科大讯飞使用指南

## 📋 概述

科大讯飞已集成到 **AI Gateway** 中，通过统一接口调用，无需单独配置客户端。

## 🚀 快速开始

### 1. 配置环境变量

在 `.env` 文件中添加：

```bash
# 科大讯飞配置
XUNFEI_API_KEY=your_appid:your_api_secret
XUNFEI_BASE_URL=https://spark-api-open.xf-yun.com/v1
XUNFEI_MODEL=lite  # 默认模型（免费）
```

### 2. 获取 API Key

1. 访问 [讯飞开放平台](https://www.xfyun.com/)
2. 注册并创建应用
3. 获取 APPID 和 APISecret
4. 组合格式：`APPID:APISecret`

### 3. 使用示例

#### Python 代码

```python
from server.ai.ai_gateway import get_gateway

# 获取 AI Gateway 实例
gateway = get_gateway()

# 方式 1：通过功能自动路由（推荐）
response = gateway.chat_completion(
    messages=[{"role": "user", "content": "分析直播氛围"}],
    function="live_analysis"  # 自动使用配置的讯飞模型
)

# 方式 2：显式指定讯飞
response = gateway.chat_completion(
    messages=[{"role": "user", "content": "你好"}],
    provider="xunfei",
    model="lite"
)

print(response["content"])
```

#### API 调用

```bash
# 通用聊天接口
curl -X POST http://localhost:9019/api/ai_gateway/chat/completions \
  -H "Content-Type: application/json" \
  -d '{
    "messages": [
      {"role": "user", "content": "你好"}
    ],
    "provider": "xunfei",
    "model": "lite"
  }'
```

## 🎯 支持的模型

| 模型 | 定价 | 特点 | 适用场景 |
|------|------|------|---------|
| **lite** | 免费 | 轻量快速 | 高频调用、测试 |
| generalv3 | ¥0.003/1K | 通用能力 | 一般任务 |
| generalv3.5 | ¥0.0036/1K | 增强能力 | 复杂任务 |
| 4.0Ultra | ¥0.005/1K | 最强能力 | 高质量要求 |

## 🔍 故障排查

### 问题 1：API Key 无效

**错误：** `401 Unauthorized`

**解决：**
1. 检查 API Key 格式：`APPID:APISecret`
2. 确认 API Key 是否激活
3. 检查是否有权限访问该模型

### 问题 2：模型不存在

**错误：** `Model 'xxx' not supported`

**解决：**
1. 检查模型名称是否正确
2. 确认账户是否有权限使用该模型
3. 使用免费的 `lite` 模型测试

### 问题 3：连接超时

**错误：** `Connection timeout`

**解决：**
1. 检查网络连接
2. 确认 base_url 是否正确
3. 尝试使用代理
