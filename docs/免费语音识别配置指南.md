## 🔥 方案一：阿里云ASR（强烈推荐）

### 为什么选择阿里云？
- ✅ **免费额度最多**: 新用户3个月免费，每月120小时
- ✅ **实时流式识别**: 支持WebSocket实时识别
- ✅ **准确率高**: 支持中文、方言，识别准确率达90%+
- ✅ **无需审核**: 开通即用，无需等待审核
- ✅ **已集成**: 本项目已内置阿里云ASR适配器

### 📝 快速配置步骤

#### 1. 开通阿里云智能语音服务

访问：https://nls-portal.console.aliyun.com/

- 登录阿里云账号（没有则注册）
- 点击"立即开通"
- 选择"按量付费"（新用户有免费额度）

#### 2. 创建项目并获取AppKey

1. 进入控制台：https://nls-portal.console.aliyun.com/applist
2. 点击"创建项目"
3. 填写项目名称（如：timao-live-asr）
4. 创建成功后，**复制AppKey**（格式：`xxxxxxxxxxxxxx`）

#### 3. 获取AccessKey

访问：https://ram.console.aliyun.com/manage/ak

1. 点击"创建AccessKey"
2. 完成安全验证
3. **复制保存**:
   - AccessKey ID（格式：`LTAI5txxxxxxxxxxxxxx`）
   - AccessKey Secret（格式：`xxxxxxxxxxxxxxxxxxxxxx`）
   
⚠️ **重要**: AccessKey Secret只显示一次，请妥善保存！

#### 4. 修改 .env 配置文件

编辑 `server/.env` 文件：

```bash
# 🆓 阿里云ASR配置（推荐）
ALIYUN_ASR_APP_KEY=你的AppKey
ALIYUN_ACCESS_KEY_ID=你的AccessKeyID
ALIYUN_ACCESS_KEY_SECRET=你的AccessKeySecret
USE_ALIYUN_ASR=1  # 启用阿里云ASR

# 禁用讯飞（已欠费）
USE_IFLYTEK_ASR=0
```

#### 5. 重启服务

```bash
cd /www/wwwroot/wwwroot/timao-douyin-live-manager
pkill -f "uvicorn.*11111"
nohup .venv/bin/python -m uvicorn server.app.main:app --host 0.0.0.0 --port 11111 --workers 1 > nohup.out 2>&1 &
```

#### 6. 验证配置

查看日志：
```bash
tail -f nohup.out | grep -i "阿里云\|aliyun"
```

应该看到：
```
✅ 阿里云ASR服务初始化成功
💰 免费额度：新用户3个月，每月120小时
```

## 🐛 常见问题

### Q1: 阿里云提示"账户余额不足"
**A**: 新用户需要先完成实名认证，才能使用免费额度。

### Q2: 提示"AppKey无效"
**A**: 
1. 确认AppKey是否正确复制（无多余空格）
2. 确认项目状态是否为"启用"
3. 稍等5分钟，配置可能需要生效时间

### Q3: 提示"签名验证失败"
**A**:
1. 检查AccessKey ID和Secret是否正确
2. 检查系统时间是否正确（签名与时间相关）
3. 确认AccessKey是否被禁用

### Q4: 如何查看已用额度？
**A**: 访问阿里云控制台 -> 费用中心 -> 用量明细
https://usagecenter.console.aliyun.com/

### Q5: 免费额度用完后怎么办？
**A**: 
1. 阿里云按量付费：约0.002元/分钟
2. 切换到腾讯云（每月50小时免费）
3. 部署本地SenseVoice模型（需要GPU）

## 💡 优化建议

### 1. 节省免费额度
- 只在需要时启动音频转写
- 及时停止不使用的服务
- 定期检查用量统计

### 2. 提高识别准确率
- 确保音频采样率为16kHz
- 使用PCM格式音频
- 避免环境噪音干扰

### 3. 降低成本
- 阿里云用完后切换到腾讯云
- 考虑部署本地模型（长期方案）
- 按需使用，避免浪费

## 🎯 推荐配置方案

**最佳实践**：同时配置多个服务商，按优先级自动切换

```bash
# .env 配置示例（多服务商）
# 优先级：阿里云 > 腾讯云 > 讯飞

# 阿里云（优先）
USE_ALIYUN_ASR=1
ALIYUN_ASR_APP_KEY=xxx
ALIYUN_ACCESS_KEY_ID=xxx
ALIYUN_ACCESS_KEY_SECRET=xxx

# 腾讯云（备用）
USE_TENCENT_ASR=0
TENCENT_SECRET_ID=xxx
TENCENT_SECRET_KEY=xxx
TENCENT_APP_ID=xxx

# 讯飞（已禁用）
USE_IFLYTEK_ASR=0
```

当阿里云额度用完后，手动修改配置：
```bash
USE_ALIYUN_ASR=0
USE_TENCENT_ASR=1
```
