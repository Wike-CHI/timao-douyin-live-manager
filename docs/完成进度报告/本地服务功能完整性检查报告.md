### 三、**缺失的关键功能** ❌

#### ⚠️ 抖音弹幕WebSocket转发服务

**当前位置**: `server/app/services/douyin_web_relay.py` (933行)

**功能描述**:
- 实时连接抖音直播间
- 抓取弹幕、礼物、点赞等消息
- 通过WebSocket推送到前端
- 支持持久化保存弹幕数据

**为什么应该在本地运行**:
1. ✅ **实时性要求高** - WebSocket连接需要低延迟
2. ✅ **计算密集** - 大量弹幕消息处理
3. ✅ **依赖本地模块** - 使用 `server.modules.douyin.liveMan`
4. ✅ **无需数据库** - 仅内存缓存或本地文件持久化
5. ✅ **不涉及用户认证** - 不需要云端用户系统

**关键代码片段**:
```python
# server/app/services/douyin_web_relay.py

from server.modules.douyin.liveMan import (
    ChatMessage,
    DouyinLiveWebFetcher,  # ← 已在 server/local/modules/douyin/
    GiftMessage,
    LikeMessage,
    ...
)

class _WebRelayFetcher(DouyinLiveWebFetcher):
    """DouyinLiveWebFetcher 的特化版本, 将消息透传给回调."""
    
    def __init__(self, live_id: str, emitter: Callable[[Dict[str, Any]], None]):
        super().__init__(live_id)
        self._emit = emitter
    
    def _emit_event(self, event_type: str, payload: Dict[str, Any]) -> None:
        """将事件推送到WebSocket客户端"""
        self._emit({"type": event_type, "data": payload})
```

**API端点**:
