## 🎓 技术要点

### 1. 相对导入层级

```python
# 同级目录
from .module import something

# 上一级目录
from ..module import something

# 上两级目录
from ...module import something
```

**示例**:
```python
# server/local/modules/ai/tip_generator.py
from ...utils.logger import get_logger  # 上三级到 server/local/utils/
```

### 2. 可选依赖处理

**原则**: 本地服务不依赖 Redis/Database/订阅服务

**方法**:
- 完整注释相关代码块
- 保留结构注释便于后期恢复
- 确保注释后缩进正确

### 3. 编码问题防范

**问题**: PowerShell `Set-Content` 破坏 UTF-8 编码

**解决**:
- 使用 `Copy-Item` 而非文本替换
- 或使用 `-Encoding UTF8` 参数
