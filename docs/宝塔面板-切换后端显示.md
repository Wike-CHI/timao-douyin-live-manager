# 宝塔面板 - 切换为后端显示模式

**审查人**: 叶维哲  
**最后更新**: 2025-11-14

## 📋 需求说明

将公网IP访问时显示的页面从**前端构建页面**切换为**后端启动成功页面**。

### 当前状态
- 访问 `http://129.211.218.135` → 显示前端构建的网页（`timao-app` 目录）

### 目标状态
- 访问 `http://129.211.218.135` → 显示后端启动成功页面（提猫直播助手服务状态）

## 🛠️ 操作步骤

### 步骤 1: 登录宝塔面板

1. 打开浏览器，访问：`http://你的服务器IP:8888`
2. 输入宝塔账号密码登录

### 步骤 2: 找到站点配置

1. 点击左侧菜单 **"网站"**
2. 在网站列表中找到 **129.211.218.135** 站点
3. 点击右侧 **"设置"** 按钮

### 步骤 3: 修改配置文件

1. 在弹出的设置窗口中，点击 **"配置文件"** 选项卡
2. 找到 `server { ... }` 块
3. 将整个 server 块替换为以下配置：

```nginx
server {
    listen 80;
    server_name 129.211.218.135;
    
    # 日志配置
    access_log /www/wwwlogs/timao-access.log;
    error_log /www/wwwlogs/timao-error.log;
    
    # 客户端最大上传大小
    client_max_body_size 50M;
    
    # 超时设置
    proxy_connect_timeout 600s;
    proxy_send_timeout 600s;
    proxy_read_timeout 600s;
    
    # 根路径：直接代理到后端（显示后端启动成功页面）
    location / {
        proxy_pass http://127.0.0.1:11111;
        proxy_http_version 1.1;
        
        # 请求头设置
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        
        # WebSocket 支持
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        
        # 禁用缓存
        proxy_buffering off;
        proxy_cache off;
    }
    
    # API 文档
    location /docs {
        proxy_pass http://127.0.0.1:11111/docs;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }
    
    location /openapi.json {
        proxy_pass http://127.0.0.1:11111/openapi.json;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
    
    # 健康检查
    location /health {
        proxy_pass http://127.0.0.1:11111/health;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        access_log off;
    }
    
    # 静态资源（后端的静态文件）
    location /static/ {
        proxy_pass http://127.0.0.1:11111/static/;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

### 步骤 4: 保存并重载

1. 点击 **"保存"** 按钮
2. 宝塔会自动验证配置是否正确
3. 如果提示配置正确，点击 **"重载配置"** 或直接关闭窗口（自动重载）

### 步骤 5: 验证效果

1. 打开新的浏览器标签页
2. 访问：`http://129.211.218.135`
3. 应该看到后端启动成功页面（紫色渐变背景，显示猫咪图标）
4. 点击页面上的链接验证：
   - **📚 API文档** → 应该打开 FastAPI 的交互式文档
   - **💚 转写状态** → 显示转写服务状态
   - **🎯 Web 界面** → 显示测试界面

## 🔄 如何恢复前端显示

如果需要重新显示前端构建的页面，修改配置文件中的 `location /` 块：

```nginx
location / {
    # 指向前端静态文件目录
    root /www/wwwroot/timao-app;
    index index.html;
    try_files $uri $uri/ /index.html;
}

# 将后端移到 /api 路径
location /api/ {
    proxy_pass http://127.0.0.1:11111/api/;
    # ... 其他proxy配置
}
```

## ⚠️ 注意事项

### 1. 后端服务必须运行

确保后端服务在 `11111` 端口运行：

```bash
# 检查后端服务状态
curl http://127.0.0.1:11111/health

# 如果无响应，启动后端服务
cd /www/wwwroot/wwwroot/timao-douyin-live-manager
python -m server.app.main
```

### 2. 端口号确认

确认后端服务端口（默认 11111）：

```bash
# 检查端口占用
netstat -tlnp | grep 11111

# 或
ss -tlnp | grep 11111
```

### 3. 防火墙设置

确保服务器防火墙允许 80 端口：

```bash
# 宝塔面板 → 安全 → 放行端口 80
```

### 4. timao-app 目录保留

`timao-app` 目录可以保留作为备份，不影响当前配置。

## 🐛 故障排查

### 问题1: 访问显示 502 Bad Gateway

**原因**: 后端服务未运行或端口不正确

**解决方案**:
```bash
# 检查后端服务
curl http://127.0.0.1:11111/health

# 查看后端日志
tail -f /www/wwwroot/wwwroot/timao-douyin-live-manager/logs/backend.log

# 重启后端服务
# (根据你的启动方式)
```

### 问题2: 配置保存失败

**原因**: Nginx 配置语法错误

**解决方案**:
```bash
# 测试配置文件
nginx -t

# 查看错误详情
```

### 问题3: 仍然显示旧的前端页面

**原因**: 浏览器缓存

**解决方案**:
1. 按 `Ctrl + Shift + R` 强制刷新
2. 或清除浏览器缓存
3. 或使用无痕模式访问

### 问题4: 样式丢失或页面显示不正常

**原因**: 静态资源路径问题

**解决方案**:
- 确保 `location /static/` 配置正确
- 检查后端 `/static` 目录是否存在

## 💡 配置说明

### 关键配置解释

```nginx
# 1. 根路径直接代理到后端
location / {
    proxy_pass http://127.0.0.1:11111;  # 后端地址
    # ...
}

# 2. WebSocket 支持（重要）
proxy_set_header Upgrade $http_upgrade;
proxy_set_header Connection "upgrade";

# 3. 禁用缓存（确保实时显示）
proxy_buffering off;
proxy_cache off;
```

### 与前端模式的区别

| 项目 | 前端模式 | 后端模式（当前） |
|------|---------|-----------------|
| 根路径 | 静态文件 | 代理到后端 |
| 目录 | `/www/wwwroot/timao-app` | 无静态目录 |
| 首页 | React 应用 | 服务状态页 |
| API | 需单独配置 | 自动包含 |
