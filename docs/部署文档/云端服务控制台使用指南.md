## 🚀 快速开始

### 1. 启动云端服务

```bash
# 使用PM2启动
pm2 start ecosystem.cloud.config.js

# 或使用快速部署脚本
./scripts/deploy_cloud.sh
```

### 2. 访问控制台

在浏览器中打开：

| 访问方式 | URL | 用途 |
|---------|-----|------|
| 本地直接访问 | `http://localhost:15000/` | 开发测试 |
| Nginx代理 | `http://localhost/` | 本地生产环境 |
| 公网访问 | `http://服务器IP/` | 远程管理 |
| 域名访问 | `https://api.timao.com/` | 生产环境（待备案） |

### 3. 测试功能

#### 测试服务连接
1. 点击页面中的"测试连接"按钮
2. 系统会自动调用 `/health` 端点
3. 弹窗显示服务状态和版本信息

#### 查看实时状态
- 状态徽章自动刷新（每30秒）
- 绿色"运行中" = 服务正常
- 红色"异常" = 服务有问题
- 灰色"离线" = 无法连接

## 📊 功能详解

### 1. 服务状态卡片

**显示信息**：
- ✅ **服务类型**: 云端服务
- ✅ **内存限制**: < 512MB（轻量级设计）
- ✅ **服务端口**: 15000（内部端口）
- ✅ **版本号**: Python版本（自动获取）

**状态徽章**：
- 🟢 **运行中**: 服务正常运行
- 🔴 **异常**: `/health` 返回非健康状态
- ⚪ **离线**: 无法连接到服务

**健康检查**：
- 点击"测试连接"按钮
- 实时调用 `/health` API
- 显示详细的服务信息

### 2. 核心功能展示

**5大核心功能**：
- 👤 **用户系统**: 注册、登录、用户信息管理
- 🔐 **身份认证**: JWT Token、会话管理
- 💳 **订阅管理**: 套餐、订阅状态、使用统计
- 💰 **支付处理**: 支付宝、微信支付集成
- ⭐ **积分系统**: 积分消耗、充值、统计

**交互效果**：
- 鼠标悬停卡片会抬升
- 边框变色提示可交互

### 3. API端点列表

**显示内容**：
- HTTP方法（POST/GET）
- API路径
- 可点击跳转

**常用端点**：
```
POST /api/auth/login         - 用户登录
POST /api/auth/register      - 用户注册
GET  /profile/basic          - 获取用户资料
GET  /api/subscription/plans - 获取订阅套餐
POST /api/payment/create     - 创建支付订单
GET  /docs                   - API文档（Swagger UI）
```

### 4. 实时监控

**自动刷新机制**：
- 每30秒自动调用 `/health` 接口
- 更新状态徽章颜色
- 无需手动刷新页面

**页面加载时**：
- 自动获取Python版本号
- 显示在"版本号"字段

## 🔐 生产环境安全

### 1. 添加访问控制

**Nginx配置**：
```nginx
location / {
    # 仅允许内网访问
    allow 192.168.1.0/24;
    allow 10.0.0.0/8;
    deny all;
    
    try_files $uri @backend;
}

location @backend {
    proxy_pass http://127.0.0.1:15000;
}
```

### 2. 基础认证

```bash
# 创建密码文件
sudo htpasswd -c /etc/nginx/.htpasswd admin

# Nginx配置
location / {
    auth_basic "Cloud Service Dashboard";
    auth_basic_user_file /etc/nginx/.htpasswd;
    try_files $uri @backend;
}
```

### 3. HTTPS加密

```bash
# 安装SSL证书
sudo certbot --nginx -d api.timao.com

# 强制HTTPS
server {
    listen 80;
    server_name api.timao.com;
    return 301 https://$server_name$request_uri;
}
```

## 🐛 常见问题

### 问题1：页面显示空白

**原因**：静态文件未找到

**解决**：
```bash
# 检查文件
ls -la server/cloud/static/index.html

# 检查日志
pm2 logs timao-cloud | grep static
```

### 问题2：Logo无法加载

**原因**：图片文件不存在或权限错误

**解决**：
```bash
# 检查Logo
ls -la server/cloud/static/assets/logo_cat_headset.jpg

# 修复权限
chmod 644 server/cloud/static/assets/logo_cat_headset.jpg
```

### 问题3：状态显示"离线"

**原因**：服务未启动或健康检查失败

**解决**：
```bash
# 检查服务
pm2 status timao-cloud

# 测试健康检查
curl http://localhost:15000/health

# 查看日志
pm2 logs timao-cloud
```

### 问题4：测试连接失败

**原因**：CORS或网络问题

**解决**：
```bash
# 检查CORS配置
# 在 server/cloud/main.py 中确认：
# allow_origins=["*"]

# 检查防火墙
sudo ufw status
sudo ufw allow 15000/tcp
```
