## 如果登录失败

### 检查账号是否存在

运行脚本检查：
```bash
python server\scripts\create_admin.py
```

脚本会：
- 如果已有管理员账号，显示现有账号信息
- 如果没有，创建新的管理员账号
- 如果用户名已存在，将该用户提升为超级管理员

### 重置密码

如果忘记密码，可以通过以下方式重置：

1. **通过管理后台**（需要其他管理员）
   - 用户管理 → 编辑用户 → 重置密码

2. **直接操作数据库**
   ```sql
   -- 需要知道用户ID或用户名
   UPDATE users SET password_hash = '新密码的哈希值' WHERE username = 'admin';
   ```

3. **重新运行创建脚本**
   - 脚本会自动将现有 `admin` 用户提升为超级管理员
