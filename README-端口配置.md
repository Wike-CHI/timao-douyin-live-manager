## ⚡ 快速开始

如果遇到502错误或无法连接，按以下步骤操作：

```bash
# 1. 开放端口（必须）
./open-ports.sh

# 2. 验证端口
./verify-ports.sh

# 3. 重启服务
./restart-all-services.sh
```

## ✅ 必做检查清单

### 服务器防火墙
- [ ] 已开放 **10050** 端口（前端）
- [ ] 已开放 **8181** 端口（后端）

```bash
# 检查端口是否监听
netstat -tulnp | grep -E ":(10050|8181)"
```

### 云服务器安全组
登录云服务商控制台，添加入站规则：

| 端口 | 协议 | 来源 | 说明 |
|------|------|------|------|
| 10050 | TCP | 0.0.0.0/0 | 前端访问 |
| 8181 | TCP | 0.0.0.0/0 | 后端API |

## ❌ 常见错误

### 502 Bad Gateway
**原因**：后端服务未启动或端口未开放  
**解决**：`./restart-all-services.sh`

### Connection Refused  
**原因**：防火墙阻止  
**解决**：`./open-ports.sh` 并检查云安全组

### ERR_CONNECTION_TIMED_OUT
**原因**：云服务器安全组未开放  
**解决**：登录云控制台配置安全组
